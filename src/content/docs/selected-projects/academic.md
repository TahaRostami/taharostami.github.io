{{< expand " Harif" "..." >}}

In almost all of Iran's universities, before the start of each semester, universities present a number of courses, and each course may be presented in a number of classes and by a number of professors. Then students must select some of these courses under some constraints, such as following a requisite and prerequisite courses chart, number of credits, etc. this leaves the students with many options to choose from.
This was **my B.Sc. project** in which I tried to propose a practical approach to automate the process of course-selection, by suggesting course schedules based on student's preferences.

**Application Language(s):** Persian

**Programming Languages and Technologies:** C#, Windows Presentation Foundation

**Member(s):** Taha Rostami

[codes and more information (Available)](https://github.com/TahaRostami/Harif)
{{< /expand >}}
